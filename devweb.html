Produire un site Web - Pokémon

(MMI)
Pour la création de ce site sur pokémon, j’ai commencé par faire un moodboard. Dans celui-ci j'y ai mis l’univers de pokémon ( terre et mer ) avec des couleurs flashy. J’y ai ajouté une carte pokemon pour mon idée de carte. On y retrouve aussi des éléments phare de pokémon comme la pokéball et le logo pokémon ainsi que les pokémon eux même j’ai aussi mis quelques bulles avec les couleurs que je souhaitais.

Pour ma première page index, pour les couleurs je souhaitais retrouver les basiques de pokémon donc le rouge, le blanc, le jaune et bleu. Pour le fond en transparence, l’image choisie représente la mer dans un monde Pokémon,et pour le filtre transparence est présent pour faire ressortir le logo ainsi que ma box connexion. En plus de ces couleurs le logo Pokémon est présent et aussi une pokéball qui est un indispensable dans Pokémon. Pour la phrase de “bienvenue”: À toi de jouer dresseur, ça fait référence au fait que pokémon est un jeu.

Pour ma deuxième page, collection. On retrouve le fond sans transparence et une bannière rouge avec comme message :”bienvenue” suite à la connexion. Pour mes carte avec les pokémon, les couleurs sont inspiré d’un livre pokémon ainsi que des carte pokemon ( contour jaune ), afin d’avoir l’impression d’une carte pokemon.
Pour la page description, j’ai repris le même design que pour les cartes de la page collection, les mêmes couleurs, le même fond afin que ça reste cohérent.
Pour les polices, c’est la même sur tout le site pour qu’il soit harmonisé. J’ai choisi cette police car il donne l’impression d’une police d’un dessin animé ou d’un jeu vidéo.

Grâce à cette SAE, j’ai pu mieux comprendre le html et aussi le php. Ça m'a permis d’approfondir mes connaissances.
